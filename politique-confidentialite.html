Politique de confidentialité

Cette page explique quelles données personnelles sont collectées via ce site et comment elles sont utilisées.

1. Données collectées

  • Formulaire de contact : nom, email, message (transmis par email).
  • Cookies techniques : uniquement si nécessaires pour le fonctionnement du site.

2. Finalité du traitement

Les données recueillies servent uniquement à répondre aux messages envoyés via le formulaire ou pour améliorer l’expérience utilisateur. Elles ne sont pas partagées à des tiers.

3. Durée de conservation

Les informations issues du formulaire sont conservées uniquement le temps nécessaire pour traiter la demande et peuvent être supprimées sur demande.

4. Sécurité

Les données ne sont pas stockées dans une base publique. Le site s’appuie sur des bonnes pratiques (HTTPS recommandé chez l’hébergeur, mots de passe sécurisés pour la boîte mail).

5. Droits des personnes

Vous disposez d’un droit d’accès, de rectification et de suppression des données vous concernant — écrivez à : t.randrianarijaona@icloud.com
